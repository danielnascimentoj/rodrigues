Sobre mim

Meu nome é Rodrigues e estou muito honrado em me apresentar como seu pré candidato a vereador. Tenho um profundo compromisso com o progresso de nossa cidade, bem como as criações de excelentes projetos, para a melhoria de toda população Messiense.

Projetos

✓ Programa Bolsa Fralda
- É uma iniciativa voltada para apoiar mães solteiras ou de baixa renda, bem como idosos, fornecendo de forma gratuíta fraldas descartáveis para quem mais precisa. Este programa visa beneficiar familias carentes de nossa cidade, garantindo que tenham acesso a um item fundamental para o cuidado de bebês e idosos.

✓ Crianças Especiais
- Criar um projeto de lei para que seja construido um espaço de inclusão social, lazer e esportivo para as crianças especiais de nossa cidade, ter um pouco mais de lazer com os seus pais e amigos.

✓ Remédio Solidário
- Essa é uma ação particular se o Rodrigues for eleito vereador, Ele fará um convênio com uma farmácia da cidade, e irá doar parte do seu salário para comprar remédios para a população carente de Messias.

✓ Fundo Esportivo
- Criar um fundo esportivo, das equipes amadoras de nossa cidade, para custeios de viagens,competições, compras de uniformes e outros materiais.

✓ Igualdade Salarial
- Este projeto visa dar igualdade salarial aos diversos profissionais de nossa cidade, em especial aos nossos professores, que exercem as mesmas funções e carga horária. Todos terão salários iguais.

✓ Lixeira na Porta
- Este projeto tem o objetivo social e de limpeza urbana, sendo instalado pela prefeitura uma lixeira padronizada em cada porta, mantendo a nossa cidade sempre limpa e agradável.

Vote pela mudança que a nossa cidade merece! Juntos, vamos construir um futuro melhor para todos.
